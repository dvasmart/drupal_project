# webform:remove:orphans

Removes orphaned submissions where the submission's webform was deleted.

#### Examples

- <code>webform:remove:orphans</code>. Removes orphaned submissions where the submission's webform was deleted.

#### Aliases

- wfro
- webform-remove-orphans
